To create PNG favicons from the SVG:

  1. Open static/Logos/naturecredits_symbol.svg in a browser or image editor
  2. Export/Save as PNG with these sizes:
    • 16x16 pixels → save as favicon-16x16.png
    • 32x32 pixels → save as favicon-32x32.png
    • 48x48 pixels → save as favicon-48x48.png
  3. Use an online tool like favicon.io to convert to .ico format
  4. Place the generated favicon.ico in the static/ folder

Or use this quick method:

Visit RealFaviconGenerator.net and upload the symbol SVG to generate all favicon formats automatically.
